About

Skapat koden i visuall studio code med programmet inbyggda git versionshantering synkat till en remote repository på github, använder parcel för all automatisering som minifiering av css, html och javascript. Optimering och formatering av bilder med avif i första hand och webp i andra med en sista redundans i jpg. Bilderna är grundbilderna ifrån en tidigare uppgift och har använt i deras grundvariant rakt ifrån kameran. All bakåtkompatabilitet sker ochså med hjälp av parcel. Anväder mig av Vercel för att hosta servern, testade med github pages men det var struligt så fort jag ville få den att köra parcel. Tänkte ladda upp en ny branch som inehöll den färdiga sidan men det vore o-elegant och förmodligen inte godtjänt.

Fin demo hemsida, med en footer!
